# GitHub for Non-Coders

> A GitHub Pages blog focused on helping everyday people use GitHub creatively — without needing to become traditional programmers.

[![ko-fi](https://ko-fi.com/img/githubbutton_sm.svg)](https://ko-fi.com/O4O41XT7AO)
---

## About

**GitHub for Non-Coders** is a [beginner-friendly blog](https://monapdx.github.io/github-for-non-coders/blog/) about using GitHub outside traditional software engineering culture.

Most GitHub tutorials assume you're trying to become a developer, contribute to enterprise software, or already understand technical jargon. This project takes the opposite approach.

**The goal is to show that GitHub can also be:**
- a publishing platform
- a creative workspace
- a documentation system
- a portfolio
- a personal archive
- a place to experiment and learn publicly

You do not need to identify as a programmer to benefit from GitHub.

## Philosophy

A lot of people are unnecessarily intimidated by GitHub.

Part of that comes from the way technology is often presented:
overcomplicated, gatekept, and full of assumptions about prior knowledge.

This blog exists because curiosity matters more than credentials.

You can learn gradually.  
You can experiment.  
You can build things before fully understanding everything.

That's how many people actually learn.

GitHub is one of the few remaining major platforms on the web that still encourages tinkering, openness, and independent creation. More people deserve access to that world.
